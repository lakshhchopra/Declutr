# Declutr Predictive Intelligence & Life Intelligence Engine Guide

The Predictive Intelligence Engine transforms Declutr into a proactive assistant ("I remembered this for you" instead of "What do you want to search?").

```
Knowledge Graph → Memory Engine → Timeline → Reverse Persona → Predictive Engine → Recommendation Planner → Approval → Action
```

## Supported Prediction Categories

1. `UPCOMING_DEADLINE`: Expiring tasks, bills, and filing dates.
2. `EXPIRING_DOCUMENT`: Passports, driver's licenses, insurance policies.
3. `UPCOMING_TRIP`: Flight bookings missing travel insurance or visa document tags.
4. `UPCOMING_MEETING`: Meeting notes referencing missing contracts or agendas.
5. `MISSING_DOCUMENT`: Incomplete context dependencies (e.g. tax filing missing W2).
6. `SUGGESTED_UPLOAD`: Recent flight ticket uploaded, suggesting insurance receipt.
7. `SUGGESTED_ORGANIZATION`: Unstructured items needing collection tags.
8. `SUGGESTED_ARCHIVE`: Aged invoices post tax season.
9. `SUGGESTED_DELETION`: Obsolete temporary screenshots.
10. `SUGGESTED_WORKFLOW`: Frequently repeated manual sequences.
11. `SUGGESTED_COLLECTION`: Grouping related travel or medical items.
12. `SUGGESTED_SUMMARY`: Long document threads requiring synthesis.
13. `RECURRING_TASK`: Weekly or monthly maintenance routines.
14. `RECURRING_EXPENSE`: Subscription renewals and bills.
15. `KNOWLEDGE_GAP`: Missing context or contradictions across notes.
16. `OPPORTUNITY_DETECTION`: Duplicate lab reports or duplicate contracts.
