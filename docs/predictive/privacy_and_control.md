# Declutr Predictive Privacy & User Controls Guide

- **100% Local Vault Privacy**: All prediction data and pattern analysis stay strictly within the user's isolated Vault. Zero telemetry leaves the platform.
- **User Control**: Users can pause learning, set confidence thresholds, toggle specific prediction categories, and dismiss predictions with feedback.
- **Safety**: Destructive actions (deletion, archival, moving data) are NEVER executed automatically. Explicit user approval is mandatory.
