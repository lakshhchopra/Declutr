# Declutr Life Operating System (LifeOS) Architecture

Declutr's LifeOS transforms the platform from a file manager into an operating system for digital life.

## Life Hierarchy Model

```
Life Area → Projects → Goals → Contexts → Knowledge → Assets
```

Everything in Declutr belongs to a meaningful life domain ("My Startup", "My Health", "My Travels", "My Finances") rather than arbitrary folder paths (`/Documents/`, `/Receipts/`).
