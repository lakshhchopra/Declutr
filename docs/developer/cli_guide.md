# Declutr CLI Guide

The **Declutr CLI** (`declutr`) allows developers to manage vaults, search content, upload files, trigger workflows, and run system diagnostics from the command line.

## Commands

- `declutr auth`: Authenticate CLI session
- `declutr vault`: List and manage vault storage
- `declutr search <query>`: Execute hybrid knowledge search
- `declutr upload <file>`: Upload document assets
- `declutr workflow`: Trigger automated workflow execution
- `declutr backup`: Initiate system snapshot backup
- `declutr diagnostics`: Check platform health status
